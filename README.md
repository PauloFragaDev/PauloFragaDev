## Proyectos propios

**[Fyndo](https://github.com/PauloFragaDev/fyndo-releases)**
App de escritorio para finanzas personales y productividad.
Tauri + Rust + Svelte. Multiplataforma (Windows / Linux / macOS). En beta privada.

**[laravel-doctor](https://github.com/PauloFragaDev/laravel-doctor)**
Analizador estático para proyectos Laravel. AST analysis, 174 tests, TUI interactiva,
autofix, análisis incremental con git diff. Disponible en Packagist.

**[trackActivity](https://github.com/PauloFragaDev/trackActivity)**
Sistema de reconstrucción de actividad laboral para facilitar el reporte en timesheets.
Daemon Python que captura señales del SO + dashboard Laravel. Offline-first.

**[simpletasks-api](https://github.com/PauloFragaDev/simpletasks-api)**
API REST con Laravel 12. Actions, Events, Notifications, Policies, Spatie QueryBuilder,
116 tests, documentación Scribe. Proyecto de referencia de arquitectura Laravel.
